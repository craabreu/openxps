A boundary condition for a dynamical variable.

Parameters:

  • lower (float) – The lower bound for the dynamical variable.

  • upper (float) – The upper bound for the dynamical variable.

  • unit (cvpack.units.units.Unit) – The unity of measurement of the bounds. If the bounds do not have a unit, use dimensionless.

class openxps.bounds.NoBounds[source]¶

Bases: Bounds

No boundary condition. The dynamical variable is allowed to take any value.

Parameters:

  • None (If it is not)

  • dynamical (its unit of measurement must be compatible with the)

  • unit. (variable's own)

Example

>>> import openxps as xps
>>> import yaml
>>> from openmm import unit
>>> bounds = xps.bounds.Periodic(-180, 180, unit.degree)
>>> print(bounds)
Periodic(lower=-180, upper=180, unit=deg)
>>> assert yaml.safe_load(yaml.safe_dump(bounds)) == bounds

class openxps.bounds.Periodic(lower, upper, unit)[source]¶

Bases: Bounds

A periodic boundary condition. The dynamical variable is allowed to wrap around the upper and lower bounds.

Parameters:

  • lower (float) – The lower bound for the dynamical variable.

  • upper (float) – The upper bound for the dynamical variable.

  • unit (cvpack.units.units.Unit) – The unity of measurement of the bounds. If the bounds do not have a unit, use dimensionless.

Example

>>> import openxps as xps
>>> import yaml
>>> from openmm import unit
>>> bounds = xps.bounds.Periodic(-180, 180, unit.degree)
>>> print(bounds)
Periodic(lower=-180, upper=180, unit=deg)
>>> assert yaml.safe_load(yaml.safe_dump(bounds)) == bounds

class openxps.bounds.Reflective(lower, upper, unit)[source]¶

Bases: Bounds

A reflective boundary condition. The dynamical variable collides elastically with the upper and lower bounds and is reflected back into the range.

Parameters:

  • lower (float) – The lower bound for the dynamical variable.

  • upper (float) – The upper bound for the dynamical variable.

  • unit (cvpack.units.units.Unit) – The unity of measurement of the bounds. If the bounds do not have a unit, use dimensionless.

Example

>>> import openxps as xps
>>> from openmm import unit
>>> bounds = xps.bounds.Reflective(1, 10, unit.angstrom)
>>> bounds == xps.bounds.Reflective(0.1, 1, unit.nanometer)
True
>>> print(bounds)
Reflective(lower=1, upper=10, unit=A)

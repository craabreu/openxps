integrators module¶

Integrators for extended phase-space simulations with OpenMM.

class openxps.integrators.BAOABIntegrator(temperature, frictionCoeff, stepSize)[source]¶

Bases: IntegratorMixin, CustomIntegrator

A BAOAB integrator.

This is a symmetric integrator.

Parameters:

  • temperature (Quantity) – The temperature of the heat bath.

  • frictionCoeff (Quantity) – The friction coefficient which couples the system to the heat bath.

  • stepSize (Quantity) – The step size with which to integrate the system.

Example

>>> import openxps as xps
>>> from openmm import unit
>>> integrator = xps.integrators.BAOABIntegrator(
...     300 * unit.kelvin, 1 / unit.picosecond, 1 * unit.femtosecond
... )
>>> integrator
Per-dof variables:
  x1
Global variables:
  a = 0.999000499833375
  b = 0.044699008184376096
  kT = 2.494338785445972
Computation steps:
   0: allow forces to update the context state
   1: v <- v + 0.5*dt*f/m
   2: constrain velocities
   3: x <- x + 0.5*dt*v
   4: v <- a*v + b*sqrt(kT/m)*gaussian
   5: x <- x + 0.5*dt*v
   6: x1 <- x
   7: constrain positions
   8: v <- v + (x-x1)/dt + 0.5*dt*f/m
   9: constrain velocities
class openxps.integrators.CSVRIntegrator(temperature, frictionCoeff, stepSize, splitting)[source]¶

Bases: IntegratorMixin, CustomIntegrator

Implements the Canonical Sampling through Velocity Rescaling (CSVR) integrator, also known as the Bussi-Donadio-Parrinello thermostat.

Parameters:

  • temperature (Quantity) – The temperature.

  • frictionCoeff (Quantity) – The friction coefficient.

  • stepSize (Quantity) – The integration step size.

  • splitting (str) – The splitting scheme. A sequence of “V”, “R”, and “O” characters representing the velocity boost, position update, and thermostat steps, respectively.

register_with_system(system)[source]¶

Register the integrator with the system.

setRandomNumberSeed(seed)[source]¶

This method overrides the openmm.CustomIntegrator method to also set the seed of the random number generator used to pick numbers from the gamma distribution. :param seed: The seed to use for the random number generator.

step(steps)[source]¶

This method overrides the openmm.CustomIntegrator method to include the efficient computation of the sum of squares of normally distributed random numbers. :param steps: The number of steps to take.

class openxps.integrators.ForceFirstMassiveGGMTIntegrator(temperature, timeConstant, stepSize, bathLoops)[source]¶

Implements a force-first, massive variant of the Generalized Gaussian Moment Thermostat (GGMT) integrator.

Parameters:

  • temperature (Quantity) – The temperature.

  • timeConstant (Quantity) – The time constant of the thermostat.

  • stepSize (Quantity) – The integration step size.

  • bathLoops (int) – The number of internal loops in the thermostat. A larger number will increase the accuracy and stability of the integrator but will also increase the computational cost.

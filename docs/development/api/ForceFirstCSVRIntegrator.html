ForceFirstCSVRIntegrator¶

class openxps.integrators.ForceFirstCSVRIntegrator(temperature, frictionCoeff, stepSize)[source]¶

Implements a force-first variant of the Canonical Sampling through Velocity Rescaling (CSVR) integrator, also known as the Bussi-Donadio-Parrinello thermostat.

Parameters:

  • temperature (Quantity) – The temperature.

  • frictionCoeff (Quantity) – The friction coefficient.

  • stepSize (Quantity) – The integration step size.

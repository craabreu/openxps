DynamicalVariable¶

class openxps.DynamicalVariable(name, unit, mass, bounds=<factory>)[source]¶

Dynamical variable for extended phase-space simulations with OpenMM.

Parameters:

  • name (str) – The name of the context parameter to be turned into a dynamical variable.

  • unit (openmm.unit.unit.Unit) – The unity of measurement of this dynamical variable. If it does not have a unit, use openmm.unit.dimensionless.

  • mass (openmm.unit.quantity.Quantity) – The mass assigned to this dynamical variable, whose unit of measurement must be compatible with dalton*(nanometer/unit)**2, where unit is the dynamical variable’s own unit (see unit above).

  • bounds (openxps.bounds.base.Bounds) – The boundary condition to be applied to this dynamical variable. It must be an instance of PeriodicBounds, ReflectiveBounds, or NoBounds (for unbounded variables). If it is not NoBounds, its unit of measurement must be compatible with the dynamical variable’s own unit.

Example

>>> import openxps as xps
>>> import yaml
>>> from openmm import unit
>>> dv = xps.DynamicalVariable(
...     "psi",
...     unit.radian,
...     3 * unit.dalton*(unit.nanometer/unit.radian)**2,
...     xps.PeriodicBounds(-180, 180, unit.degree)
... )
>>> dv
DynamicalVariable(name='psi', unit=rad, mass=3 nm**2 Da/(rad**2), bounds=...)
>>> dv.bounds
PeriodicBounds(lower=-3.14159..., upper=3.14159..., unit=rad)
>>> assert yaml.safe_load(yaml.safe_dump(dv)) == dv

Returns a Lepton expression representing the distance between this extra degree of freedom and another variable.

Parameters:

other (CollectiveVariable | Self) – The other variable to which the distance will be calculated.

Returns:

A string representing the distance between the two variables.

Return type:

str

Example

>>> import openxps as xps
>>> import pytest
>>> from openmm import unit
>>> dv = xps.DynamicalVariable(
...     "psi0",
...     unit.radian,
...     3 * unit.dalton*(unit.nanometer/unit.radian)**2,
...     xps.PeriodicBounds(-180, 180, unit.degree)
... )
>>> psi = cvpack.Torsion(6, 8, 14, 16, name="psi")
>>> dv.distanceTo(psi)
'(psi-psi0-6.28318...*floor(0.5+(psi-psi0)/6.28318...))'
>>>

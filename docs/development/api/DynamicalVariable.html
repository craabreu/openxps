DynamicalVariable¶

class openxps.DynamicalVariable(name, unit, mass, bounds)[source]¶

An OpenMM context parameter turned into a dynamical variable.

Parameters:

  • name (str) – The name of the context parameter to be turned into a dynamical variable.

  • unit (Unit) – The unity of measurement of this dynamical variable. It must be compatible with the MD unit system (mass in dalton, distance in nanometer, time in picosecond, temperature in kelvin, energy in kilojoules_per_mol, angle in radian). If the collective variables does not have a unit, pass dimensionless.

  • mass (Quantity | Real) – The mass assigned to this dynamical variable, whose unit of measurement must be compatible with dalton*(nanometer/unit)**2, where unit is the dynamical variable’s own unit (see above).

  • bounds (Bounds | None) – The boundary condition to be applied to this dynamical variable. It must be an instance of openxps.bounds.Periodic, openxps.bounds.Reflective, or None (for unbounded variables).

Example

>>> import math
>>> import openxps as xps
>>> import yaml
>>> from openmm import unit
>>> dv = xps.DynamicalVariable(
...     "psi",
...     unit.radian,
...     3 * unit.dalton*(unit.nanometer/unit.radian)**2,
...     xps.bounds.Periodic(-math.pi, math.pi),
... )
>>> dv
<psi in Periodic(-3.14..., 3.14...), unit is rad, mass is 3 nm**2 Da/(rad**2)>
>>> serialized_dv = yaml.safe_dump(dv)
>>> print(serialized_dv)
!openxps.DynamicalVariable
bounds: !openxps.bounds.Periodic
  lower_bound: -3.14...
  upper_bound: 3.14...
  version: 1

  version: 1

>>> yaml.safe_load(serialized_dv)
<psi in Periodic(-3.14..., 3.14...), unit is rad, mass is 3 nm**2 Da/(rad**2)>
